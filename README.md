# bamazon

# First run the file by typing "node bamazonCustomer.js"
![alt text](https://github.com/michaelosgood/bamazon/blob/master/images/one.png)

# All products, departments, prices, and stock will display
![alt text](https://github.com/michaelosgood/bamazon/blob/master/images/two.png)

# Type in the name of the product you would like to purchase 
![alt text](https://github.com/michaelosgood/bamazon/blob/master/images/three.png)

# Type in the quantity of the product you would like to purchase
![alt text](https://github.com/michaelosgood/bamazon/blob/master/images/four.png)

# You'll receive a "Product Purchased!" notification and the table will reload with udpated quantities
![alt text](https://github.com/michaelosgood/bamazon/blob/master/images/five.png)

# You can repeat the steps to purchase more products or press "Q" to quit
![alt text](https://github.com/michaelosgood/bamazon/blob/master/images/five.png)
